## Default Permission

Default permissions for the push notification plugin

#### This default permission set includes the following:

- `allow-check-permission`
- `allow-request-permission`
- `allow-get-token`
- `allow-get-or-create-keys`
- `allow-reset-keys`
- `allow-set-context`
- `allow-get-launch-notification`
- `allow-register-listener`
- `allow-remove-listener`
